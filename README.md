# NvimFiles

## Installing

You will need [git](https://git-scm.com/downloads) and [nvim](https://neovim.io/)

Clone into your `$HOME\.config` directory or `~\.config`

```bash
git clone https://git@github.com:andreneji/nvim.git
```
